One of the things I was told before I got my first Mac was that Apple's user interfaces were the most intuitive. Back in the early 2000s, I think that was probably true (compared, for instance, to Microsoft Windows, where you'd tap the "Start" button to shut it down).

This morning when I was sending an email and wanted to put my initials in italics, which is my wont, and I typed ⌘I, I was surprised to see the word "Oblique" appear in the toolbar.

![Screenshot showing Apple Mail in macOS Tahoe 26.5 Beta (25F5068a)][screenshot]

Meanwhile in the icon-embellished "Format" menu, I is for Italic as nature intended.

Surely this is a regression and will be fixed before the next release?
